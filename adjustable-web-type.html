Developer documentation.

A playful drop-in script for exploring letterforms — deform real reference font outlines by default, with preset axes (bubbliness, serif length, width). Parametric anatomy handles and static compare modes are optional. See third-party font licenses.

lib/sculpt.js

Install & mount

1

Load the script, get a global

Add <script src="lib/sculpt.js"></script>. Exposes window.SculptLettering = { SandboxWordmark, DeformableOutlineWordmark, AnatomyDeformWordmark, OutlineWordmark, createWordmark, Glyph, glyphs, presets, … }. (Wordmark is kept as a back-compat alias for SandboxWordmark.) It also works as a CommonJS / ES module export of the same object.

2

Construct, mount, optionally make it interactive

const wm = new SculptLettering.SandboxWordmark('hello', {
  // Hand-authored Bézier sandbox; preset is optional and only the
  // `none` preset routes here in practice. Use createWordmark() to
  // dispatch by preset.pipeline.
  color:    '#2a2ae5',
  tracking: 12,
  padding:  64,
});
wm.mount(document.getElementById('stage'));   // or selector string
wm.makeInteractive();                         // draggable handles + tooltip

Allowed text is [A-Za-z ]; any other character throws "No glyph registered for character '…'". Filter input upstream.

Wordmark API

setText(newText)

Position-matched diff. Glyphs at indices that still match the old text keep their tuned params; the rest are re-instantiated from the preset. Use this instead of constructing a new wordmark to preserve user edits across keystrokes.

makeInteractive() / freezeInteraction()

Toggles handles + hover tooltip. Tooltip styling, edge-flip, and pointer-leave fallbacks are built in — no CSS required.

enableMouseFollow(opts?) / disableMouseFollow()

Maps the page mouse position into tangent-handle drag deltas so the wordmark morphs as the cursor moves (default tangentOnly: true — curvature, bowl tension, terminal arms; not positional handles). Snapshot is captured on enable and restored on disable. Options: origin ('center' default | 'topleft'), strength (default 0.4), clamp (default 220), tangentOnly (default true; set false for wild mode).

resetAll() / resetGlyph(index)

Reverts each glyph to the active preset via resolvePresetParams (or module defaults when preset is none). Re-renders.

toSVG() → string

Static SVG markup of the current state, with no handle layer. Safe to drop into HTML, an <img src> data URL, or a build artifact.

toState() → object

Serializable snapshot: { text, tracking, color, padding, preset, glyphs: [{character, params}], modes: { mouseFollow } }. JSON-safe. preset is the preset key (e.g. 'bubbly') so exported bundles can re-instantiate new letters from the same mood. When mouse-follow is on, params contains the rest values (the snapshot taken at enable), not the cursor-morphed live values.

await toInteractiveBundle() → string

Full self-contained HTML document — library source inlined, no CDN, no fetches. Carries the full state from toState(), including any active modes: if mouse-follow is on when you call this, the embed boots with mouse-follow on too. Returns the document string; paste it anywhere. If the library was inlined (not loaded by <script src>), set SculptLettering.SandboxWordmark.LIBRARY_SOURCE first.

Glyphs & handles

Glyph parameters

Each character has its own param set with min/max ranges. Curated glyphs (a n o s h i e t r l w d b c m g) expose hand-authored Bezier params (bowl width, tension, aperture, serifLength on serif presets, optical bearings, …). Remaining letters use monoline schema: xHeight/capHeight, width, strokeWeight, slant, curvature. Inspect with wm.glyphs[i].paramRanges.

Handle types

Filled circles are positional anchors — dragging moves the param along a screen axis. Hollow circles with arms are tangent params (e.g. bowlTopTension, curvature) — the arm length encodes the value; a small square marks the tangent anchor. Programmatic equivalents: wm.glyphs[i].set(name, value) or setMany({…}), both clamped to range.

Reference face & render modes

Deformable outline (default)

Default for font presets — loads the OFL reference face and deforms paths with preset axes. Requires SculptLettering.setOpentypeParser(opentype). 

SculptLettering.setOpentypeParser(opentype);
const wm = await SculptLettering.createWordmark('Hello', {
  presetKey: 'bubbly',
  renderMode: 'outline',
});
wm.mount('#stage');
wm.makeInteractive();
wm.setAxis('bubbliness', 0.6);

Parametric letters (fallback / pedagogy)

Choose Parametric letters (no reference font) or renderMode: 'parametric'. Hand-authored anatomy handles; presets seed defaults and glyphParams only. 

{
  name: 'bubbly',
  fontRef: 'Rubik Bubbles',
  defaults:    { strokeWeight: 28, curvature: 0.7, /* … */ },
  glyphParams: { a: { bowlWidth: 60, /* … */ } },
}

Registering a custom glyph

Pass a module to SculptLettering.registerGlyph({ character, defaultParams, paramRanges, construct(p), handles(p), advance(p), bounds(p), tangentParams? }). construct returns an array of SVG path d strings; bounds returns {minX,maxX,minY,maxY} in the glyph's local coordinates (baseline at y=0).
