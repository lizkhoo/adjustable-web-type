Developer documentation.

A drop-in script that draws an alphabet of parametric letterforms with draggable Bezier handles. One global, two classes, five font-mood presets. Everything you need to embed it is on this page.

Install & mount

1

Load the script, get a global

Add <script src="lib/sculpt.js"></script>. Exposes window.SculptLettering = { Wordmark, Glyph, glyphs, presets, registerGlyph, getRegisteredGlyphs }. It also works as a CommonJS / ES module export of the same object.

2

Construct, mount, optionally make it interactive

const wm = new SculptLettering.Wordmark('hello', {
  preset:   SculptLettering.presets.bubbly,  // optional
  color:    '#0a0a0b',
  tracking: 12,
  padding:  64,
});
wm.mount(document.getElementById('stage'));   // or selector string
wm.makeInteractive();                         // draggable handles + tooltip

Allowed text is [A-Za-z ]; any other character throws "No glyph registered for character '…'". Filter input upstream.

Wordmark API

setText(newText)

Position-matched diff. Glyphs at indices that still match the old text keep their tuned params; the rest are re-instantiated from the preset. Use this instead of constructing a new wordmark to preserve user edits across keystrokes.

makeInteractive() / freezeInteraction()

Toggles handles + hover tooltip. Tooltip styling, edge-flip, and pointer-leave fallbacks are built in — no CSS required.

enableMouseFollow(opts?) / disableMouseFollow()

Maps the page mouse position into every handle's drag-delta function, so the whole wordmark morphs as the cursor moves. Snapshot is captured on enable and restored on disable. Options: origin ('center' default | 'topleft'), strength (default 0.4), clamp (max pixel offset, default 220), tangentOnly (false).

resetAll() / resetGlyph(index)

Reverts params to the glyph module's defaultParams (not the preset). Re-renders.

toSVG() → string

Static SVG markup of the current state, with no handle layer. Safe to drop into HTML, an <img src> data URL, or a build artifact.

toState() → object

Serializable snapshot: { text, tracking, color, padding, glyphs: [{character, params}], modes: { mouseFollow } }. JSON-safe. When mouse-follow is on, params contains the rest values (the snapshot taken at enable), not the cursor-morphed live values.

await toInteractiveBundle() → string

Full self-contained HTML document — library source inlined, no CDN, no fetches. Carries the full state from toState(), including any active modes: if mouse-follow is on when you call this, the embed boots with mouse-follow on too. Returns the document string; paste it anywhere. If the library was inlined (not loaded by <script src>), set SculptLettering.Wordmark.LIBRARY_SOURCE first.

Glyphs & handles

Glyph parameters

Each character has its own param set with min/max ranges. Curated glyphs (a n o s h i e t r l w d) expose hand-authored Bezier params (bowl width, tension, aperture, terminal arm, …). The other 40 letters share a monoline schema: xHeight/capHeight, width, strokeWeight, slant, curvature. Inspect with wm.glyphs[i].paramRanges.

Handle types

Filled squares are positional anchors — dragging moves the param along a screen axis. Hollow circles with arms are tangent params (e.g. bowlTopTension, curvature) — the arm length encodes the value. Programmatic equivalents: wm.glyphs[i].set(name, value) or setMany({…}), both clamped to range.

Presets & extending

Preset shape

{
  name: 'bubbly',
  defaults:    { strokeWeight: 28, curvature: 0.7, /* … */ },
  glyphParams: { a: { bowlWidth: 60, /* … */ }, /* … */ },
}

For each character, glyphParams[ch] wins if present; otherwise the defaults block is filtered against the glyph's own defaultParams keys and applied. Unknown keys are silently dropped, so one defaults block can cover the whole alphabet.

Registering a custom glyph

Pass a module to SculptLettering.registerGlyph({ character, defaultParams, paramRanges, construct(p), handles(p), advance(p), bounds(p), tangentParams? }). construct returns an array of SVG path d strings; bounds returns {minX,maxX,minY,maxY} in the glyph's local coordinates (baseline at y=0).
